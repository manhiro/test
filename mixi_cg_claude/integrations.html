社内ツール連携ガイド

MCP（Model Context Protocol）を使うと、Claude Code が Slack や Notion と直接やりとりできるようになります。
ミーティングメモの自動まとめ、Notion ドキュメントの読み書きが会話形式で。

Step 0 — まず確認

Mixi 社内での許可について

MCP は外部のサービスへアクセスする機能です。会社のセキュリティポリシーによって利用可能範囲が異なります。以下を確認してから進めてください。

✅ 通常は許可不要
  • 自分のワークスペースの Slack を読む
  • 自分がアクセスできる Notion を読む
  • ローカルファイルの読み書き
  • Web 検索（Brave Search MCP など）
⚠️ 情シスに確認推奨
  • 全社向け Slack チャンネルへの投稿
  • 社内データベースへの接続
  • 外部 API の利用（本番環境）
  • 自動投稿・自動書き込みの設定
⚠️
まずは読み取り専用から始めることを強く推奨します。Slack を「読む」だけなら問題が起きにくく、情シスの承認も得やすいです。
Tool 01
💬
Slack MCP
チャンネルを読む・メッセージをまとめる

できること

議事録まとめ
#制作MTGの今日の会話をまとめて、決定事項とTODOを抽出して
✓ 決定事項: 3件 / TODO: 5件 をまとめました
情報収集
#技術情報チャンネルで今週出たHoudiniに関する話題をまとめて
✓ 5件のトピックを発見・要約しました

セットアップ手順

  • 1
    Slack App を作成する
    api.slack.com/apps にアクセス → 「Create New App」→「From scratch」→ アプリ名と対象ワークスペースを設定。
  • 2
    Bot Token Scopes を設定する
    「OAuth & Permissions」→「Bot Token Scopes」に以下を追加：
    channels:read channels:history groups:read groups:history users:read
  • 3
    ワークスペースにインストールする
    「Install to Workspace」をクリック → 「Bot User OAuth Token」（xoxb-...）をコピーして保存。
  • 4
    MCP の設定ファイルを作成する
    ホームディレクトリ（ ~/.claude/ ）に mcp_servers.json を作成：
{ "mcpServers": { "slack": { "command": "npx", "args": ["-y", "@modelcontextprotocol/server-slack"], "env": { "SLACK_BOT_TOKEN": "xoxb-your-token-here", "SLACK_TEAM_ID": "T00000000" } } } }
✅
claude を再起動すると Slack MCP が使えるようになります。「#チャンネル名の最新メッセージをまとめて」と話しかけてみましょう。
Tool 02
📄
Notion MCP
ドキュメントを読む・週報を自動生成する

できること

ドキュメント読み込み
プロジェクト仕様ページを読んで、未完了のタスクをリストアップして
✓ 7件の未完了タスクを発見しました
自動ドキュメント更新
今週の制作ログをもとに、進捗ページを更新して
✓ 進捗ページに3セクションを追記しました

セットアップ手順

  • 1
    Notion Integration を作成する
    notion.so/my-integrations にアクセス → 「New integration」→ 名前（例：Claude Code）を入力 → 「Submit」。
  • 2
    Internal Integration Token をコピー
    「Internal Integration Token」（ntn_...）をコピーして安全な場所に保存。
  • 3
    アクセスしたいページに Integration を追加
    Notion の対象ページを開く → 「…」→「Connections」→「Claude Code」を追加。
  • 4
    MCP 設定に追加する
{ "mcpServers": { "slack": { /* 上記のSlack設定 */ }, "notion": { "command": "npx", "args": ["-y", "@notionhq/notion-mcp-server"], "env": { "OPENAPI_MCP_HEADERS": "{\"Authorization\": \"Bearer ntn-your-token\"}" } } } }
✅
claude を再起動して「Notionの〇〇ページを読んで」と話しかけてみましょう。
