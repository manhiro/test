準備

Claude Code を使えるようにする
— 最初の1回だけ、10分で完了

STEP 1
Node.js をインストール
nodejs.org から LTS 版をダウンロードして「次へ」を押し続けるだけ。
node --version
STEP 2
Claude Code をインストール
ターミナルを開いて以下を貼り付けてEnter。
npm install -g @anthropic-ai/claude-code
npm install -g @anthropic-ai/claude-code
STEP 3
起動 → ブラウザでログイン
以下を実行するとブラウザが開きます。claude.ai アカウントでログイン。
claude
✅
ログインして Claude> というプロンプトが表示されたら準備完了。あとは下のプロンプトを貼り付けるだけです。
初回だけ実行する

最初にこれを貼ると、
Claude が環境を整えてくれる

以下のプロンプトを Claude に貼り付けると、Claude が質問しながらあなた専用の環境を作ります。設定ファイルも、フォルダも、コマンドも — 全部 Claude が作ってくれます。

🚀
初回1回だけ
環境セットアップウィザード
Claude が質問しながら、あなた専用の環境を丸ごとセットアップしてくれます
① 役割・使用ツール・作業フォルダを聞いてくれる → ② グローバル設定ファイル（CLAUDE.md）を作成 → ③ 役割に合ったコマンドを3つ自動生成 → ④ 「今日は何をしますか？」で終了
貼り付けた後は Claude の質問に答えるだけ。設定ファイルの作成場所や内容はすべて Claude が決めてくれます。
📄
プロジェクト開始時CLAUDE.md
プロジェクト設定ファイルを自動生成
Claude が質問して、そのプロジェクト専用の CLAUDE.md を作ってくれます
① 6つの質問に答える → ② CLAUDE.md の内容を提案 → ③ 確認・修正 → ④ 保存。次回からこのフォルダで起動すると Claude が内容を覚えた状態でスタート
このプロンプトは作業フォルダで Claude を起動してから使ってください。そのフォルダに CLAUDE.md が作られます。
毎日使う

毎回のセッションを
このプロンプトで始める

Claude を起動したら、まずこれを貼り付けましょう。前回の続きから再開できます。

📁
毎日使う
プロジェクト開始ウィザード
新規 or 既存を選ぶだけで、Claude がプロジェクト管理を全部やってくれる
新規 → フォルダ作成・CLAUDE.md 生成・ディレクトリ構成を提案 ／ 既存 → プロジェクト一覧表示・前回の状況を要約して報告
このプロンプトは初回セットアップが済んでいると、作業フォルダの場所を Claude が覚えています。
📝
毎日使う
作業終わりの記録・引き継ぎ
今日の作業内容を自動まとめ → 明日の自分への引き継ぎメモを作成
今日変更したファイルを確認 → 作業内容を要約 → 引き継ぎメモを docs/daily/ に保存 → CLAUDE.md の「最終作業」を更新
毎日これで終わると、翌日「プロジェクト開始ウィザード」を使ったとき Claude が昨日の状況を正確に教えてくれます。

デザイナー
