Section 01

Claude Code って
何ができるの？

AIに日本語で話しかけるだけで、面倒な作業を肩代わりしてくれるツールです。
プログラミングの知識はいりません。

📁
フォルダを丸ごと理解
プロジェクト全体を読んで、何がどこにあるか把握。整理・リネームの提案まで。
📝
ドキュメントを自動作成
シーン設定・シェーダー仕様・作業ログを、そのまま読みやすい文書に変換。
⚙️
スクリプトを生成・修正
「こういう処理をしたい」と伝えるだけで VEX・Python・MEL を書いてくれる。
🔍
技術調査・まとめ
Blender / UE5 の新機能やベストプラクティスを調べ、チーム向けにまとめる。
🔁
繰り返し作業を自動化
毎週同じ形式のレポート、アセットのチェック作業——一度設定すれば後は任せるだけ。
💬
会話しながら作業
「もう少しシンプルに」「英語でも書いて」と追加指示も自然な日本語でOK。
💡
Claude Code はターミナル?（黒い画面）で動きますが、操作は会話形式です。コマンドをほとんど覚える必要はありません。
Section 02

3ステップで
セットアップ完了

Windows でも Mac でも同じ手順です。所要時間：約 10 分。

Step 1 — Node.js をインストール

Claude Code を動かすために必要なソフトウェアです。Node.js とは？?

🌐
nodejs.org にアクセスして LTS（推奨版） をダウンロードしてインストールしてください。インストーラーの「次へ」を押し続けるだけでOKです。

インストール後、ターミナルで確認：

PS>$ node --version
✅
v18.x.x 以上が表示されれば成功です。
Step 2 — Claude Code をインストール

ターミナルで以下のコマンドを実行します。コピーボタンを押すとクリップボードに入ります。

PS>$ npm install -g @anthropic-ai/claude-code
⚠️
エラーが出た場合、会社のプロキシ設定が必要な場合があります。情シスに確認してみてください。

インストール確認：

PS>$ claude --version
✅
バージョン番号が表示されれば成功です。
Step 3 — 認証する

Claude Code を使うには認証が必要です。2 つの方法があります。APIキーとは？?

★ おすすめ
claude.ai サブスクリプション
Pro / Max プランならすぐ使えます。
ブラウザでログインするだけ。
API KEY
API キーで認証
従量課金型。console.anthropic.com でキーを発行します。

起動して認証：

PS>$ claude
✅
ブラウザが開いてログイン画面が表示されます。claude.ai のアカウントでログインしてください。完了したらターミナルに戻ると会話が始まります。
Section 03

最初のプロンプトを
入力してみよう

プロンプト?とは AI への指示文のこと。日本語で自然に話しかけて OK です。

作業したいフォルダに移動して起動：

PS> cd C:\Users\YourName\Documents\MyProject
PS> claude
$ cd ~/Documents/MyProject
$ claude

起動したら日本語で話しかけるだけ：

You › このフォルダの構成を説明してください
Claude › このプロジェクトには以下のファイルが含まれています…
（フォルダ構成の詳しい説明が表示されます）
💡
終了は /exit と入力するか Ctrl+C を押してください。

Section 05

もっと使いやすく
カスタマイズする

CLAUDE.md? というファイルをプロジェクトフォルダに置くと、Claudeが毎回チームの前提を知った状態でスタートします。

プロジェクトフォルダに CLAUDE.md を作成して書くだけ：

# プロジェクト概要 MIXI CG映像プロジェクト。Houdini + UE5で制作。 # チーム・ツール 制作: 3名（CG×2、ディレクター×1） 使用ツール: Houdini 20.5 / UE5.4 / After Effects 2024 # ファイル命名規則 assets/textures/[asset]_[type]_v001.png # 返答スタイル - 専門用語は日本語で説明を添えること - コードには必ず使い方のコメントを入れること
💡
このファイルがあるフォルダで claude を起動すると、毎回この内容を読んでから会話が始まります。いちいち前提を説明する手間がなくなります。

よく使うコマンド一覧

コマンド説明
/help使えるコマンドの一覧を表示
/clear会話履歴をクリアして新しいセッションを開始
/exitClaude Code を終了
Ctrl + C現在の処理を中断
↑ キー前に入力したプロンプトを呼び出す
/compact長くなった会話を要約して続ける（コスト削減）
🔌
さらに高度な連携（Slack・Blender・データベースとの接続）は MCP（Model Context Protocol）? を使います。まずは基本に慣れてから。
