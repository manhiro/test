## まず試してほしい3つのプロンプト

### 1. 今週の技術トレンドをキャッチアップ
```
Blender・Houdini・Unreal Engine 5の直近3ヶ月の
主要アップデートと、CGアーティストが注目すべき
新機能をまとめてください。
各ツールごとに「即使える機能」と「中長期で重要な変化」
に分けて教えてください。
```

### 2. プロジェクトのリファレンス方針を言語化
```
[プロジェクト名]: ○○
[ジャンル/スタイル]: ○○
[参考にしたい作品/監督/スタジオ]: ○○

上記プロジェクトのビジュアル方針をまとめてください。
- ライティングの特徴
- カラーグレーディングの傾向  
- カメラワーク・レンズの特性
- 避けるべきNGスタイル
```

### 3. ベテランの手順をドキュメント化
```
以下の作業手順をチーム向けのマニュアルに変換してください。

[箇条書きや口頭メモをそのまま貼り付ける]

出力形式:
- 目的と前提条件
- ステップごとの手順（スクリーンショット挿入箇所も明示）
- よくあるミスとその対処
- 応用・発展Tips
```

## Claude Code を使う場合（開発者向け）

このリポジトリ内で `claude` コマンドを使うと、
ファイル操作・スクリプト生成・自動化まで対応できます。

```bash
# プロジェクト用リサーチを自動実行
claude "今週のHoudini SideFX公式ブログの新記事をまとめてレポートを作って"
```
