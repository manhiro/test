# 技術キャッチアップ プロンプト集

CG映像制作に関わるツール・技術のトレンドを素早く把握するためのプロンプト集。

---

## ツール別 最新情報収集

### Blender
```
Blender の最新バージョン（4.x系）で追加された機能のうち、
映像制作（VFX・アニメーション・モーショングラフィックス）に
直接影響する変更点を教えてください。

特に以下の観点で整理してください：
1. レンダリング（Cycles/EEVEE）の変化
2. ジオメトリノードの新機能
3. アニメーション・リグ関連
4. コンポジット・後処理
5. パフォーマンス改善

各項目に「実務でどう使えるか」を1行で添えてください。
```

### Houdini
```
Houdini 20.x の新機能のうち、VFX・シミュレーション・
プロシージャルモデリングの現場で役立つものを教えてください。

以下の優先度で整理してください：
【高】 すぐ業務に取り入れられる
【中】 学習コスト低めで3ヶ月以内に習得可能
【低】 将来的に重要だが今はウォッチだけでよい
```

### Unreal Engine 5
```
Unreal Engine 5（最新バージョン）のCG映像制作向け機能を
まとめてください。

Sequencer・Lumen・Nanite・MetaHumanに絞り、
実写合成・バーチャルプロダクション用途での
実用性と注意点を教えてください。
```

### AI生成ツール（映像×AI）
```
映像制作・CG制作の現場で実際に使われているAIツールを
以下のカテゴリで整理してください：

1. テクスチャ・マテリアル生成（Stable Diffusion, Adobe Firefly等）
2. モーション生成（Cascadeur, DeepMotion等）
3. VFXシミュレーション補助
4. 音楽・SE生成
5. 動画生成（Sora, Runway, Kling等）

各ツールに「商用利用OK/要確認/NG」を明記してください。
```

## 競合・業界動向リサーチ

### スタジオ技術調査
```
[調査対象スタジオ名] の最近の作品で使われている技術・
パイプラインについて、公開情報（インタビュー・
SIGGRAPH発表・メイキング）からわかることをまとめてください。

知りたいこと：
- 主要ツール構成
- 独自開発している技術
- ワークフローの特徴
- 採用・育成の方針
```

### SIGGRAPH / FMX 論文キャッチアップ
```
SIGGRAPH [年] で発表された論文・プレゼンのうち、
実務のCG制作パイプラインに3〜5年以内に影響しそうな
研究を5つ挙げてください。

各研究について：
- 概要（2〜3行）
- 実装・製品化の見通し
- 現時点でできる準備
```
