ターミナルとは？

コマンドで操作する「テキスト版のFinderやエクスプローラー」
💡 たとえるなら
Finder（Mac）やエクスプローラー（Windows）は、フォルダをクリックして操作しますよね。
ターミナルは同じことをテキストで操作する画面です。

クリックの代わりに「cd Documents（Documentsに移動）」のように文字を打って操作します。
Claude Code もここで起動して、会話形式で使います。

あなたのOSでの開き方

🪟 Windows
PowerShell を使います。

① スタートボタンを右クリック
② 「Windows PowerShell」または「ターミナル」を選択
🍎 Mac
ターミナル を使います。

① Spotlight（⌘+Space）を開く
② 「ターミナル」と入力してEnter

知っておくと便利な基本操作

  • 1
    claude と入力してEnter → Claude Code が起動します
  • 2
    ↑ キー → 前に入力したコマンドを再呼び出しできます
  • 3
    Ctrl + C → 処理を中断・キャンセルします
  • 4
    cls（Windows）/ clear（Mac）→ 画面をリセットします
✅
Claude Code は起動後は会話形式なので、ターミナルの操作を覚える必要はほとんどありません。claude と打って起動するだけでOKです。
