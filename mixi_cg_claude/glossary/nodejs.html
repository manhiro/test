用語解説

Node.js とは？

Claude Code を動かすために必要な「エンジン」
💡 たとえるなら
Blender を動かすには Windows や macOS が必要なように、
Claude Code を動かすには Node.js が必要です。

Node.js 自体を使いこなす必要はまったくありません。
インストールするだけで OK。あとは Claude Code が勝手に使ってくれます。

インストール手順

  • 1
    nodejs.org にアクセスする
  • 2
    LTS（推奨版） のボタンをクリックしてダウンロード
    ※「Current」ではなく「LTS」を選んでください
  • 3
    ダウンロードしたインストーラーを開き、「次へ」を押し続けてインストール完了
  • 4
    ターミナルを開いて確認：
    $ node --version
    v20.x.x のようにバージョンが表示されれば成功です

npm とは？

Node.js をインストールすると npm（Node Package Manager） も一緒に入ります。Claude Code のインストールには npm install -g @anthropic-ai/claude-code というコマンドを使いますが、これは npm を使って Claude Code を取得するコマンドです。

npm は「ソフトウェアのダウンロード専用コマンド」と考えてください。App Store や Microsoft Store のCLI版のようなものです。

⚠️
インストール後はターミナルを一度閉じて再起動すると、node コマンドが正しく認識されます。
✅
Node.js のバージョンが v18 以上であれば Claude Code は動作します。最新版で問題ありません。
