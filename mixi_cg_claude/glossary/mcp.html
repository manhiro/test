MCP とは？

Model Context Protocol — Claude Code を外部ツールと繋ぐ「プラグイン」
💡 たとえるなら
Blender のアドオンと同じです。
標準の Blender に「アドオンを追加」すると新しい機能が使えるように、
Claude Code に MCP サーバーを追加すると、Slack・データベース・ファイルサーバーなど外部ツールと連携できるようになります。

MCP でできること（例）

💬
Slack 連携
「#制作チャンネルの最新メッセージをまとめて」と Claude に依頼できる
🗄️
データベース接続
プロジェクト管理DBを直接読んでレポートを生成
🌐
Web 検索
最新の技術情報をリアルタイムで検索して回答
📁
クラウドストレージ
Google Drive や S3 上のファイルを直接読み書き
🎨
Figma 連携
Figma のデザインデータを読み込んでドキュメント化
📊
Notion / Linear
タスク・ドキュメント管理ツールと双方向連携

どうやって動く？

👤
あなた
日本語で指示

🤖

⇄
🔌
MCP サーバー
外部ツール

Claude Code が必要に応じて MCP サーバー経由で外部ツールにアクセスします。あなたは普通に日本語で指示するだけです。

📌
MCP の設定には JSON ファイルの編集が必要です。まずは基本の Claude Code に慣れてから挑戦しましょう。
詳細は discovery.html の「MCP」カテゴリを参照してください。
✅
最初のうちは MCP なしで十分です。テキストでの会話・ファイル読み書き・スクリプト生成だけで、日常業務の大半はカバーできます。
